The Project

The idea behind this research project is to predict Zika epidemics using information available in social networks such as Twitter. This is motivated by the idea that people that interact frequently in social networks might interact in real life.

An Example:

The following figure shows an interactive figure of the Twitter network we have been able to reconstruct so far. Each node represents one person and the connection between nodes represents an interaction. Through the study of these interactions we can better understand how people relate to each other and how diseases spread through these interactions.

The figure can be zoomed in, out and dragged!
